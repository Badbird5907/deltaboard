# Deltaboard

Deltaboard is a TKL mechanical keyboard.

## Features:
- TKL Layout
- Kailh MX hotswap sockets
- OLED display
- Integrated RP2040
- USB-C
- RGB lighting
- EC11 Encoder
- QMK/VIA firmware
